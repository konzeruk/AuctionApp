{
	Данный проект является библиотекой классов, которая является вспомогательной для всех сервисов и клиента.

	-----------------------------------------------------------------------------------------
	Папки:
		
		ContextDB: хранит классы с контекстом бд

		Extensions: хранит классы-расширения

		HttpClients: хранит классы с HttpClient, который общения между сервисами и клиентом

		Models: хранит модели данных
			DTO: хранит модели данных, которые отправляются клиенту
			Entities: хранит сущности данных, которые достаются из бд
		
		Repositories: хранит интерфейсы репозиториев бд
}
